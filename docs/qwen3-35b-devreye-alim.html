Qwen3.6-35B-A3B-FP8 Model Devreye Alım Dökümanı

Tarih: 21 Nisan 2026  |  Hazırlayan: TAI – Yapay Zeka Ekibi  |  Sunucu: plusaigpu01

1. Amaç ve Kapsam

Bu döküman, Qwen3.6-35B-A3B-FP8 büyük dil modelinin plusaigpu01 sunucusundaki GPU 7 (NVIDIA H200 144G) üzerine kurulumu, yapılandırılması ve servise alınması sürecini adım adım açıklamaktadır.

Model, vLLM OpenAI-uyumlu API sunucusu üzerinden 8015 portu ile hizmete açılmış; mevcut chatboss Docker ağına dahil edilmiştir.

5. Docker Compose Yapılandırması

5.1 Dizin Yapısının Hazırlanması

$ mkdir -p /data01/kurulum/qwen3-35b $ cd /data01/kurulum/qwen3-35b

5.2 docker-compose.yml Dosyası

Aşağıdaki yapılandırma dosyası oluşturulmuştur. Model GPU 7'ye sabitlenmiş, port 8015 olarak atanmış ve gpu-memory-utilization değeri GPU'daki mevcut kullanıma göre 0.48 olarak ayarlanmıştır.

name: qwen3-35b networks: chatboss: external: true x-deploy-7: &gpu-7-deploy deploy: resources: reservations: devices: - driver: nvidia device_ids: [ '7' ] capabilities: [ gpu ] services: qwen3-35b: image: vllm/vllm-openai:latest volumes: - /data02/models/Qwen3.6-35B-A3B-FP8:/models/Qwen3.6-35B-A3B-FP8:ro command: > --model /models/Qwen3.6-35B-A3B-FP8 --served-model-name Qwen3.6-35B-A3B --gpu-memory-utilization 0.48 --tensor-parallel-size 1 --max-model-len 32768 --host 0.0.0.0 --port 8000 ports: - "8015:8000" <<: *gpu-7-deploy ipc: host networks: - chatboss
gpu-memory-utilization: 0.48 — GPU 7'de başka işlemler ~70 GB VRAM kullandığından, kalan alan için bu oran belirlenmiştir. Bu değer KV önbelleği için 28.33 GiB ayırarak 370.656 token ve yaklaşık 40x eşzamanlı istek kapasitesi sağlamaktadır.

6. vLLM Docker İmajının Güncellenmesi

Qwen3.6-35B-A3B-FP8 modeli eski vLLM sürümleriyle (v0.11.0) desteklenmemektedir. Gerekli mimari (Qwen3_5MoeForConditionalGeneration) için vLLM'in son sürümü indirilmiştir.

$ docker pull vllm/vllm-openai:latest latest: Pulling from vllm/vllm-openai ... Status: Downloaded newer image for vllm/vllm-openai:latest docker.io/library/vllm/vllm-openai:latest

13. Kullanım Notları

Servis yeniden başlatmak gerektiğinde aşağıdaki komutlar kullanılmalıdır:

$ cd /data01/kurulum/qwen3-35b $ docker compose down $ docker compose up -d $ docker compose logs -f

API üzerinden modeli sorgulamak için OpenAI uyumlu herhangi bir istemci kullanılabilir. Base URL olarak http://plusaigpu01:8015/v1 adresi girilmeli, model adı olarak Qwen3.6-35B-A3B seçilmelidir.

Thinking modu hakkında: Model varsayılan olarak çıktı öncesinde İngilizce düşünce bloğu (<think>...</think>) üretmektedir. Bu arayüzde gösterilmek istenmiyorsa ya /no_think sistem mesajı olarak eklenmeli ya da API yanıtında <think> blokları filtrelenmelidir.
